# BlueMoney
Proyecto final del curso de "Fundamento de programación en JavaScript" para el segundo trimestre del año 2021.

## Pasos luego de clonar el proyecto
```
Primero ve hacia la consola en Visual Studio Code (View/terminal)
```
```
Debes ejucar en consola el comando **npm install** esto instalara todas las dependencias y creara la
carpena node_module
```
```
Luego ejecutas el comando **npm run build** para que genere la carpeta app donde se encuentra el
bundle de los .js
```
```
Para correr el proyecto ejecutas el comando **npm run dev** este iniciara el servidor y cuando hagas
un cambio y guardes, automaticamente lo subira y actualizara la pagina
```
